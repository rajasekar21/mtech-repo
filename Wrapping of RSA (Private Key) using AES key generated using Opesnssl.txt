First on HSM we have to create a partition and initilize it via luna client and change the partition policy to export insted of cloning. 

Doc :- https://www.thalesdocs.com/gphsm/luna/7/docs/network/Content/admin_partition/policies/cloning_key_export.htm?Highlight=export%20mode#KeyExportMode

Create a AES key using the command "openssl rand -out aeskey.bin 32" on /usr/safenet/lunaclient/bin location.

And take a material ouput via "xxd -p aeskey.bin", I take that output and import that AES key material in with the key name "AESTest". 
And change the key usages as wrap and unwrap.
